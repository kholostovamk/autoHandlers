If the images and links are not aligned properly on a big screen, you can use CSS media queries to adjust the layout based on the screen size. For example, you can set different CSS rules for screens larger than a certain size.

Here's an example of how you can use media queries to adjust the layout:

css
/* default CSS */
.tipimg {
  float: left;
  margin-right: 10px;
}
.tiplink {
  display: block;
  margin-bottom: 10px;
}

/* CSS for screens larger than 768px */
@media (min-width: 768px) {
  .tipimg {
    width: 50%;
    float: left;
    margin-right: 20px;
  }
  .tiplink {
    display: inline-block;
    margin-bottom: 0;
    vertical-align: top;
  }
}

In the above example, the default CSS rules apply to screens smaller than 768px, and the media query applies to screens larger than 768px. Within the media query, we've adjusted the width of the image to 50%, increased the margin-right to 20px, and changed the display property of the link to inline-block to align it properly with the image. The vertical-align: top property ensures that the links are aligned to the top of the images.

You can adjust the values in the media query to suit your specific needs.
